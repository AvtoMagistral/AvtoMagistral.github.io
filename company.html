Перелік дорожньо-будівельних робіт

Товариство з обмеженою відповідальністю «АвтоМагістральБуд» – компанія з великим досвідом (більше 14 років) в галузі дорожньо-будівельних робіт. ТОВ «АвтоМагістральБуд» має можливість виконати весь спектр дорожньо-будівельних та ремонтних робіт будь-якої складності на I-IV категорії доріг:
✓ влаштування дорожньої основи;
✓ влаштування покриття з асфальтобетону;
✓ установлення бортових каменів;
✓ установлення дорожних бар’єрів;
✓ будівництво тимчасових доріг;
✓ капітальний та поточний ремонт існуючих доріг.

Крім того, успішно виконуємо:
✓ земляні роботи:
✓ розробка грунту;
✓ копання котлованів та траншей;
✓ планування майданчиків;
✓ влаштування насипів;
✓ укріплення відкосів;
✓ комплекс робіт з благоустрою територій:
✓ благоустрій водойм;
✓ влаштування підпірних стінок;
✓ монтаж водовідводів та колодязів;
✓ укладання тротуарної плитки та природного каменю;
✓ видалення дерев та корчування пнів;
✓ встановлення огорож;
✓ прибирання територій.
✓утримання доріг загального користування та міських вулиць і доріг.
